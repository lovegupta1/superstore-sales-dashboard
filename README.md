# 📊 Superstore Sales Dashboard

An interactive Power BI dashboard analyzing retail sales data across regions, segments, and categories with a 15-day sales forecast.

## 🔍 Overview
- **Total Sales:** 1.6M | **Orders:** 22K | **Profit:** 175K | **Avg Ship Days:** 4

## 📌 Key Insights
- **West region** leads with 33% of total sales
- **Consumer segment** dominates at 48%
- **Office Supplies** top category at 0.64M revenue
- **Standard Class** shipping most used (0.33M)
- **15-day sales forecast** built using Power BI analytics

## 🛠️ Tools Used
- Power BI Desktop
- DAX (Data Analysis Expressions)
- Sales forecasting module

## 📂 Dashboard Pages
1. **Sales Overview** – Region, Segment, Category, Payment Mode breakdown
2. **Sales Forecast** – 15-day future sales prediction
